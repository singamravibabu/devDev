# Sting Operators
- Text joining (concatenating) strings
- Repetition operator

## Concatenation operator
- `+` sign for text joining
- Syntax:
```
str1 + str2 + str3 + str4 + ...
```

## Repetition operator
- `*` repetition operator
- Syntax:
```
str * int
```

- `chr()` for converting a number to a Unicode character
